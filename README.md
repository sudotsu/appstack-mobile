## üöÄ Quick Start

### Option 1: PWA Deployment (15 minutes)

```bash
# Install dependencies
npm install

# Deploy to Vercel
npx vercel --prod

# Install on Android:
# 1. Open URL in Chrome on phone
# 2. Menu ‚Üí "Install app"
# 3. Done!
```

### Option 2: Native Android APK (2-3 hours)

```bash
# 1. Build for export
npm run build

# 2. Add Android platform
npx cap add android

# 3. Sync assets
npx cap sync android

# 4. Build APK
cd android && ./gradlew assembleDebug

# APK will be at: android/app/build/outputs/apk/debug/app-debug.apk
```
